# 🐍 Radha Naam Jap Game

A devotional twist on the classic Snake game 🪷
Grow your snake by chanting the divine name **Radha**.

---

## 🎮 Features

* 🧘 Naam Jap based gameplay
* 🐍 Snake grows with each “Radha” collected
* 🔤 Username appears on the snake
* 📱 Fully responsive (works on mobile & desktop)
* 🎯 Score + Best Score tracking
* ✨ Smooth animations and aesthetic UI

---

## 🕹️ How to Play

* Use **Arrow Keys / WASD** to move
* Eat the **Radha** blocks to grow
* Avoid hitting walls or your own tail
